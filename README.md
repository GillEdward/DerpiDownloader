# DerpiDownloader

需要注册Derpi账号并获取个人API-KEY

你可以在账户里找到你的API-KEY ：https://trixiebooru.org/registrations/edit or https://derpibooru.org/registrations/edit

脚本访问的是https://trixiebooru.org/

内容与主站完全一致，且不需要挂梯子

有网络连接造成的的进度卡死时请重启程序（这是有概率发生的事）

* 有Python可以使用.py文件

* 无Python可以使用经过pyinstaller打包过的.exe文件
